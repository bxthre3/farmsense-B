# Contributing to FarmSense

Thank you for your interest in contributing to FarmSense! We welcome contributions from the community to help make deterministic farming more accessible and powerful.

## How to Contribute

1. **Report Bugs**: Use GitHub Issues to report bugs. Please include steps to reproduce and your environment details.
2. **Suggest Features**: We love new ideas! Open an issue to discuss potential features.
3. **Submit Pull Requests**:
   - Fork the repository.
   - Create a new branch for your feature or fix.
   - Ensure your code follows the project's style and passes all tests.
   - Submit a PR with a clear description of your changes.

## Code of Conduct

Please be respectful and professional in all interactions within the FarmSense community.
